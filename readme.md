# TG 99 Bot

## Update requirements

```sh
pip freeze > requirements.txt
```

## First install

```sh
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Initialize and start

```sh
source .venv/bin/activate
python app.py
```

## (All-In-One) Launch via Docker Compose

```sh
docker compose up --build
```
